## 实现要点

采用**逐维度串行处理**架构，从最低维到最高维依次启动 Triton kernel：

1. **Host 侧维度循环**: `for dim_idx in range(ndim - 1, -1, -1)`，每次处理一个维度的 repeat
2. **双路径 grid 分发**:
   - Small Grid (`total_blocks <= VEC_CORE_NUM`): 2D grid `(outer_size, num_inner_blocks)`
   - Large Grid (`total_blocks > VEC_CORE_NUM`): 1D grid + 标量循环分配
3. **`r` 声明为 `tl.constexpr`**: 触发编译期 loop unroll，消除 repeat 循环开销
4. **BLOCK 按 2 的幂次分级**: 64/128/256/512/1024/2048/4096/8192，使 `num_inner_blocks` 尽量小
5. **int32 索引**: `tl.program_id` 和偏移量均显式转 `tl.int32`，避免 int64 标量降级
